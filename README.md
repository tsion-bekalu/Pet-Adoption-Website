Pawsome – Pet Adoption Website
Pawsome is a clean and friendly pet adoption website designed to help users explore available pets, learn about the adoption process, and connect with the organization. The website has multiple pages, each designed to give users clear information in a simple layout.

Pages
  -Home
    Hero section
    Experience counters
    Why Choose Us
    How Adoption Works
    Success Stories
  -About
    Short introduction about the organization
    Mission and vision
    Our Story
  -Available Pets
    List of available dogs, cats, and other pets
    Basic details about each pet
    Adoption call-to-action buttons
  -Contact
    Contact email
    Phone number
  -Become a Volunteer
    Information about volunteering
    Benefits and responsibilities
    Volunteer sign-up button

Built With
  HTML5
  CSS3
  Flexbox & Grid
  Google Fonts

How to Use
  Download or clone the project
  Keep all files in the same folder
  Open any page (example: index.html) in a browser
  Use the navigation bar to move through each page
  No backend or server setup is required.

What We Learned

  How to structure content in sections

  How color consistency improves the UI
